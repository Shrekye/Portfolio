# Portfolio d'Erwann Dumas

## Description

Ce portfolio présente mon parcours en cybersécurité et rassemble mes expériences, formations, compétences, projets ainsi que mes moyens de contact.

Le portfolio est accessible en ligne : [https://Shrekye.github.io/Portfolio/](https://Shrekye.github.io/Portfolio/)

## Structure du projet

- `index.html` : Page principale du portfolio.
- `css/styles.css` : Fichier de styles personnalisés.
- `js/` : Scripts JavaScript utilisés pour les interactions.
- `pics/` : Images et ressources graphiques.
- `cv/` : Version PDF du CV.

## Modifications à faire / Personnalisation

Pour adapter ce portfolio à vos besoins, vous pouvez modifier :  

1. **Nom et informations personnelles**  
   - Mettre à jour le nom, l’introduction et les sections de contact dans `index.html`.

2. **Expériences et formations**  
   - Ajouter, supprimer ou modifier les expériences professionnelles et académiques.

3. **Compétences et projets**  
   - Ajouter vos compétences techniques ou soft skills.  
   - Décrire vos projets en ajoutant des images ou des liens externes.

4. **Styles et apparence**  
   - Modifier les couleurs, polices et mises en page dans `css/styles.css`.  
   - Ajouter des animations ou effets via `js/` si nécessaire.

5. **Images et médias**  
   - Remplacer les images dans le dossier `pics/` par vos propres visuels.

6. **CV**  
   - Remplacer le PDF dans le dossier `cv/` par votre CV à jour.

## Modifications avancées / Personnalisation supplémentaire

1. **Internationalisation (multilingue)**  
   - Ajouter des versions en anglais, espagnol, etc., avec un sélecteur de langue.  
   - Prévoir des fichiers `index_en.html`, `index_es.html` ou un système de traduction JS.

2. **SEO et métadonnées**  
   - Ajouter des balises meta (`title`, `description`, `og:image`) pour un meilleur référencement et partage sur les réseaux sociaux.

3. **Section blog ou actualités**  
   - Créer une page dynamique pour partager articles, tutos ou réalisations récentes.

4. **Intégration de services externes**  
   - Formulaire de contact fonctionnel via EmailJS, Formspree ou autre.  
   - Liens vers GitHub, LinkedIn, Twitter ou portfolio de projets (Dribbble, Behance…).

5. **Animations et interactions avancées**  
   - Scroll animations avec AOS.js ou GSAP.  
   - Effets sur hover, carrousels de projets, lightbox pour images.

6. **Optimisation performance**  
   - Compression des images et minification des CSS/JS.  
   - Utilisation d’un favicon et d’icônes SVG optimisées.

7. **Accessibilité (a11y)**  
   - Ajouter des attributs `alt` aux images.  
   - Vérifier les contrastes de couleurs et navigation clavier.

8. **Versioning et mise à jour facile**  
   - Prévoir des fichiers JSON ou YAML pour les données (expériences, projets) afin de ne modifier que le contenu sans toucher au HTML/CSS.
